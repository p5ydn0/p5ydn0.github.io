VBA学习之三：常用代码模块

3. 常用代码模块

3.1. Windows与Sheet激活与选中

Windows("name").Activate
Sheet("name").Select

3.2. 利用vlookup函数查询

Set raw data = Workbooks("name.xls").Worksheets("name").Cells
data = Application.Vlookup(Cells(i ,j), raw data,number, 0)

3.3. 选择单元格和行列

Range(Cells(1, 1), Cells(4, 2)).Value = 12
Rows(3).Select  ' 选择了整行
Columns(3).Select  ' 选择整列

3.4. 界面显示

  • 插入“用户窗体”模块；
  • 打开工作表示时自动打开用户界面：

在ThisWorkbook中写入：

Private Sub Workbook_Open()
    Home(“用户窗体”模块的name).Show
End Sub
  • 设置按钮打开界面：

新建一个模块，然后新建个Sub，Sub中写入Home.Show，然后将按钮的宏指向该Sub就可以了。

  • 界面中运行的相关代码，直接双击该界面，然后在其中编辑相关代码即可。

  • 在界面中编辑的大概方法是这样的：新建个用户窗体模块之后，就会有个界面窗口，然后可以在其中插入各种控件；每次插入一个控件，就可以选中该空间，然后就可在属性窗口（需设置将其显示出来）中对其进行如名称更改等操作；同时你也就知道该控件的名称了。然后双击该控件，就能到达该控件所对应的代码区进行相关的操作。同时，也可以在将光标放在某个控件Sub里面之后，在代码编辑器的右上角选择更多的控件“动作”。

后续对控件的相关操作可以通过相关的调用代码来完成；如：Home.WIN_statusDisplay.Text = "Hello, World!"（其中，Home是该用户窗体模块的名称；WIN_statusDisplay是某个字符串显示控件的名称。

3.5. 函数返回多个值

函数定义：

Function One(...) As Variant

    Dim result(1 TO 5) As Interger

    result(1) = number1
    result(2) = number2
    ...

    One = result

End Function

调用函数：

Dim column
column = One()
column(1) = number3
...

3.6. 写入文档

Dim OutputFilePath As String
Dim Content As String

' 文字档案位置
OutputFilePath = ThisWorkbook.Path & "\email.txt"  ' 注意email前要有\

' 开启OutputFilePath文字档，使用编号#1档案代码
Open OutputFilePath For Output As #1

' 要写入档案的内容
Content = "hello, world!"
Print #1, Content

' 关闭编号#1档案
Close #1
