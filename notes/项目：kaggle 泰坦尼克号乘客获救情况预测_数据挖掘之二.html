项目：kaggle 泰坦尼克号乘客获救情况预测

数据挖掘之二：初步分析数据

先看一下每个/多个属性和最后的 Survived 之间有着什么样的关系。

1. 乘客各属性的分布情况

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
data_train = pd.read_csv('data/train.csv', engine='python')

fig1 = plt.figure(num=1, figsize=(28,18))

# 先看一下总体的获救情况
ax1 = fig1.add_subplot(2,3,1)
data_train['Survived'].value_counts().plot(kind='bar')
plt.xlabel('Survived')
plt.ylabel('number of people')
plt.title('Survived situation(1 is rescued)')

# 再看一下客舱等级的分布情况
ax2 = fig1.add_subplot(2,3,2)
data_train['Pclass'].value_counts().plot(kind='bar')
plt.xlabel('Pclass')
plt.ylabel('number of people')
plt.title('Passenger grade distribution')

# 接下来看一下 Age 与 Survived 之间是否有相关性
# 可以看出年龄对是否获救也是有一定影响的
ax3 = fig1.add_subplot(2,3,3)
plt.scatter(data_train['Age'], data_train['Survived'])
plt.xlabel('Age')
plt.ylabel('Survived situation(1 is rescued)')

# 接下来看一下不同 Pclass 所对应的 Age 的密度图
ax4 = fig1.add_subplot(2,3,4)
data_train['Age'][data_train['Pclass']==1].plot(kind='kde', label='class1')
data_train['Age'][data_train['Pclass']==2].plot(kind='kde', label='class2')
data_train['Age'][data_train['Pclass']==3].plot(kind='kde', label='class3')
plt.xlabel('Age')
plt.ylabel('Density')
plt.title('age and rescued situation')
plt.legend(loc='best')

# 接下来看一下登陆港口的分布情况
ax5 = fig1.add_subplot(2,3,5)
data_train['Embarked'].value_counts().plot(kind='bar')
plt.xlabel('Embarked')
plt.ylabel('number of people')
plt.title('embarked situation')

# 接下来看一下性别与是否获救是否有相关性
ax6 = fig1.add_subplot(2,3,6)
data_train['Survived'][data_train['Sex']=='female'].value_counts().plot(kind='bar')
plt.xticks([0, 1],['female', 'male'])

运行后可得到如下图表：

从这些图表中我们可以粗略的得到一些信息和有一些猜想。

从输出的图形可以看到：

被救的人 300 多点，不到半数；3 等舱乘客比 1、2 等舱要多很多；遇难和获救的人年龄似乎跨度都很广；3 个不同的舱年龄总体趋势似乎也一致，2/3 等舱乘客 20 岁多点的人最多，1 等舱 40 岁左右的最多；登船港口人数按照 S、C、Q 递减，而且 S 远多于另外俩港口；。

此时可能就会有一些想法：

  • 不同舱位/乘客等级可能和财富/地位有关系，最后获救概率可能会不一样；
  • 年龄、性别对获救概率也一定是有影响的；
  • 获救情况也可能和登船港口有关系。

这里只是很简单的猜想，接下来做进一步的分析。

2. 各属性与获救结果的关联统计：

获取不同等级的获救情况：

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
data_train = pd.read_csv('data/train.csv', engine='python')

fig2 = plt.figure(num=2, figsize=(28,18))

# 将 Pclass 中没有获救的筛选出来，然后统计其中各个 class 的个数
Survived_0 = data_train['Pclass'][data_train['Survived']==0].value_counts()
Survived_1 = data_train['Pclass'][data_train['Survived']==1].value_counts()

print(Survived_0)
print('--------')
print(Survived_1)

df = pd.DataFrame({'Rescued':Survived_1, 'Not Rescued':Survived_0})
print(df)

ax1 = fig2.add_subplot()
df.plot(kind='bar', stacked=True)
plt.xlabel('Pclass')
plt.ylabel('number of people')
plt.title('class and rescued')

获取不同性别及不同等级的获救情况：

由于根据上面的初步分析（其实很大原因也是根据我们的直观经验猜想），我们可以看到（猜想）性别和客舱等级（尤其是是不是 3 等级）会对获救情况有一定的相关性，所以我们重点看一下这两个因素对获救情况的影响。

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

ax1.set_xticklabels(['rescued', 'not rescued'], rotation='horizontal')
