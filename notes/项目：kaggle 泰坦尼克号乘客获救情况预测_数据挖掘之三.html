特征工程(feature engineering)非常非常的重要。

1. 特征量缺失值处理

从之前得到的信息可知：

PassengerId    891 non-null int64
Survived       891 non-null int64
Pclass         891 non-null int64
Name           891 non-null object
Sex            891 non-null object
Age            714 non-null float64
SibSp          891 non-null int64
Parch          891 non-null int64
Ticket         891 non-null object
Fare           891 non-null float64
Cabin          204 non-null object
Embarked       889 non-null object

缺失的特征有 Age、Cabin 和 Embarked。

先来解决 Cabin 和 Age 丢失数据的问题；有丢失数据对下一步的工作影响非常大。

对于 Cabin 特征：

Cabin 缺失的很多，暂时就按之前的处理方式，即 Cabin 的有无来进行处理；将这个属性的值设为 Yes 和 No 两种类型。

通常遇到缺值的情况，处理的方式有：

  • 如果 缺值的样本占总数比例极高 ，那么可能直接 舍弃 就可以了；如果将其作为作为特征加入，可能反倒带入 noise，影响最后的结果；

  • 如果 缺值的样本适中 ，而该属性为 非连续值特征属性 (比如说类目属性)，那么就把 NaN 作为一个新类别加到类别特征中；

  • 如果 缺值的样本适中 ，而该属性为 连续值特征属性 ，那么可能会考虑给定一个 step (比如这里的 age，我们可以考虑每隔 2/3 岁为一个步长)，然后把它离散化，之后把 NaN 作为一个 type 加到属性类目中；

  • 如果 缺失的值个数不是特别多 ，那可以试着根据已有的值拟合一下数据，然后补充上。

对于 Age：

在此，上面所述的后两种处理方式应该都是可行的；在此先采用最后一种方法将数据拟合补充上(这可能不是一个好的方法)。

在此使用 scikit-learn 中的 RandomForest 来拟合缺失的年龄数据。

代码如下：

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.ensemble import RandomForestRegressor

data_train = pd.read_csv('data/train.csv', engine='python')


# 使用RandomForestClassifier填补缺失的年龄属性
def set_missing_age(df):
    # 把已有的数值型特征取出来传入Random Forest Regressor中
    age_df = df[['Age', 'Fare', 'Parch', 'SibSp', 'Pclass']]
    # 将乘客分成已知年龄和未知年龄两部分
    known_age = age_df[age_df['Age'].notnull()].values
    unknown_age = age_df[age_df['Age'].isnull()].values

    y = known_age[:, 0]
    X = known_age[:, 1:]

    rfr = RandomForestRegressor(random_state=0, n_estimators=2000)
    rfr.fit(X, y)

    predictedAges = rfr.predict(unknown_age[:, 1:])

    age_predict = df['Age'][df['Age'].isnull()] = predictedAges

    return df, rfr


set_missing_age(data_train)
print(data_train)

2. 特征量因子化处理

接下来要做的是将类目型特征变量因子化(因为在用逻辑回归建模的时候，需要输入的特征变量都是数值型的值)。

在此使用 pandas 中的 get_dummies() 来完成这个工作，并将因子化后的列表拼接在原来的 data_train 之上。

从当前 data_train 的数据可以看出，可能需要进行因子化处理的特征为 Sex、Cabin、Embarked、Pclass。需要注意的是，Pclass 也进行了因子化处理。因为尽管 Pclass 只要 3 个值，但是这三个值的大小代表的不是数值大小，而是种类，所以也进行了因子化处理。

# 因子化处理
dummies_Cabin = pd.get_dummies(data_train['Cabin'], prefix='Cabin')
dummies_Sex = pd.get_dummies(data_train['Sex'], prefix='Sex')
dummies_Embarked = pd.get_dummies(data_train['Embarked'], prefix='Embarked')
dummies_Pclass = pd.get_dummies(data_train['Pclass'], prefix='Pclass')

# 将因子化后得到的 Dataframe 与原始的 Dataframe 进行拼接
df = pd.concat(
    [data_train, dummies_Cabin, dummies_Embarked, dummies_Sex, dummies_Pclass],
    axis=1)

# 将不需要的列“丢弃”掉；
df.drop(['Pclass', 'Name', 'Sex', 'Ticket', 'Cabin', 'Embarked'],
        axis=1,
        inplace=True)
# print(df)

从输出结果可以看出，达到了预期的效果。

3. 特征量归一化处理

如果仔细观察 Age 和 Fare 两个属性就会发现，这两个属性的值的变化范围很大；这会对模型的收敛速度有很大影响，所以要先进行归一化处理。

# 归一化处理
from sklearn import preprocessing

scaler_age = preprocessing.scale(df['Age'])
df['Age'] = scaler_age
scaler_fare = preprocessing.scale(df['Fare'])
df['Fare'] = scaler_fare

从输出结果可以看出，Age 和 Fare 特征变量被归一化了，达到了预期的效果。
